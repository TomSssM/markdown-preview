Queue

Queue is something and here comes the beautiful explanation and everything about it to get the learner going with code samples and so on and so forth you know

class Queue {
  constructor() {
    this.collection = [];
  }

  get front() {
    return this.collection[0];
  }

  get size() {
    return this.collection.length;
  }

  print() {
    return [...this.collection];
  }

  enqueue(element) {
    return this.collection.push(element);
  }

  dequeue() {
    return this.collection.shift();
  }

  isEmpty() {
    return this.size === 0;
  }
}

Example

const queue = new Queue();
queue.enqueue('a');
queue.enqueue('b');
queue.enqueue('c');
queue.print(); // => ['a', 'b', 'c']
queue.dequeue(); // => 'a' <- ['b', 'c']
queue.front; // => 'b'
queue.print(); // => ['b', 'c']
